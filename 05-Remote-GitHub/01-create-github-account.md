
# 👤 Create GitHub Account

---

## 🎯 Why This Matters

GitHub is where:

- your code lives online
- teams collaborate
- projects are shared
- open-source happens

Without GitHub, Git is limited to local usage.

---

## 🧠 Core Idea

> GitHub = cloud platform for Git repositories

## 🧩 Steps to Create Account

---

### 1. Go to GitHub

Visit:

👉 [https://github.com](https://github.com)

---

### 2. Click "Sign Up"

---

### 3. Enter Details

* email
* password
* username

---

### 4. Verify Email

GitHub sends verification link

---

### 5. Choose Plan

* Free (recommended)
* Paid (optional)

---

### 6. Complete Setup

You now have a GitHub account 🎉

**Q: What is GitHub?**

Answer:

> GitHub is a cloud-based platform that hosts Git repositories and provides collaboration tools like pull requests, issue tracking, and version control for teams.
