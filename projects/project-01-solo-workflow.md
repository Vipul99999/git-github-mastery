
# 🧑‍💻 Project 01: Solo Developer Workflow

> “Master Git alone before working in a team.”

---

## 🎯 Objective

Build a feature using proper Git workflow.

---

## 🧪 Tasks

* Create repo
* Create feature branch
* Make commits
* Merge into main

---

## ⚙️ Steps

```bash
git init
git switch -c feature-login

echo "login code" > login.txt
git add .
git commit -m "feat: add login"

git switch main
git merge feature-login
```

---

## 🧠 Concepts Practiced

* commits
* branching
* merging

---

## 🏁 Outcome

```text
You can manage your own workflow cleanly
```
